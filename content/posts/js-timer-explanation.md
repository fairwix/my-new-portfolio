1. Переменная `x` создаётся в момент вычисления `x = Date.now()` внутри параметров стрелочной функции. В неё сохраняется текущее время в миллисекундах (Unix-время) непосредственно перед запуском цикла. Конкретное число зависит от того, когда запустили код, поэтому заранее его не узнать. `x` нужна как точка отсчёта: в цикле из текущего времени вычитается `x`, и пока разница меньше заданного значения, программа висит.

2. IIFE (Immediately Invoked Function Expression) — это функция, которая объявляется и тут же вызывается. В оригинале она позволяет выполнить код немедленно и не засорять глобальную область видимости. Тот же смысл можно передать без IIFE, например так:

```javascript
console.time('a');
const y = 2 ** 14;   // 16384
const x = Date.now();
while (Date.now() - x < y);
console.timeEnd('a');
```

Этот фрагмент работает идентично: запускается таймер, фиксируются начальное время и длительность, затем пустой цикл ждёт нужное количество миллисекунд, и в конце выводится время.

Результат выполнения: цикл длится примерно 16384 миллисекунды, что равно 16,384 секунды. Целое количество секунд, которое показывает console.timeEnd — 16.
